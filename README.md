# RulerJs

The RulerJS library exported as [Node.js](https://nodejs.org/) modules.

## Installation

Using npm:
```shell
$ npm i -g rulerjs
$ npm i --save rulerjs
```

In Node.js:
```js
// Load the full build.
var ruler = require('rulerjs');



## Support

Tested in Chrome 54-55, Firefox 49-50, IE 11.
