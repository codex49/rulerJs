# RulerJs

The RulerJS library exported as [Node.js](https://nodejs.org/) modules.

![build status](https://img.shields.io/travis/hajjiTarik/objArr/master.svg?style=flat-square)


## Installation

Using npm:
```shell
$ npm i -g rulerjs
$ npm i --save rulerjs
```

In Node.js:
```js
// Load the full build.
var ruler = require('rulerjs');
